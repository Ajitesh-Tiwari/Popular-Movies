Introduction

Android application build using The Movie Database ( TMDb ) API and hosted on Google Play Store. Displays 'Most Popular' and 'Top Rated' movies along with details ( Plot, Rating, Reviews, Trailers, etc. ). Supports Multi Pane tablet view and sharing of movies on other platforms.

Feature List

  • Review and watch trailer of movies from all around the world with ratings.
  • Get to know their release date and story line.
  • Movies are sorted as 'Most Popular' or 'Top Rated' by default to show you only the best.
  • Share these movies among your friends and save to your favorite list.
  • Multi pane support for tablets.
